# AI Terminal Chatbot 

This is a simple terminal-based chatbot built using Python and the OpenAI API.
The program allows users to type messages in the terminal and receive AI-generated responses.

The purpose of this project is to demonstrate basic API integration with Python and how a simple conversational chatbot can be created.

## Requirements

* Python 3.8 or higher
* OpenAI Python library

Install the required library:

pip install openai

## Setup

1. Open the file `ChatBot_AI_new.py`.
2. Replace the placeholder API key with your own:

api_key = "YOUR_API_KEY_HERE"

For security reasons, the API key is not included in the code.

## Running the Program

Run the chatbot using:

python ChatBot_AI_new.py

Type your message in the terminal and the chatbot will respond.
To exit the program, type **exit**.

## Note

You must provide your own OpenAI API key to use the chatbot.
